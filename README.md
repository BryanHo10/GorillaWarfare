### Setup

Install Nodejs and npm [here](https://nodejs.org/en/).

Install Webpack
```
cd GorillaWarfare
npm install webpack --save-dev
npm install webpack-cli --save-dev
```

Load Dependencies into your Project
```
npm install
```

### Build
Bundle and build code with:
```
npm run build
```
